1. Скачать нужные библиотеки через терминал

	pip install flask flask-sqlalchemy flask-login flask-mail werkzeug

2. Зайти через среду разработки в папку Baza и ввести в терминале

   Python app.py

3. Откройте браузер и перейдите по адресу: http://127.0.0.1:5000 (также сама среда разработки должна выдавать ссылку)
